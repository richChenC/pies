# PIES 软件使用说明

## 1. 软件定位

PIES 用于从 Excel 或业务数据文件夹中提取探头使用记录，并完成以下工作：

- 去重
- 统计
- 异常提示
- 历史累计
- 图表展示
- 表格与图表导出

软件处理原则：

1. 默认只读源数据，不修改原始文件
2. 当前批次与历史累计分开管理
3. 图表与表格基于统一统计结果生成，避免口径不一致

## 2. 主界面功能

### 2.1 文件处理

- `处理文件夹`
  - 读取包含业务文件的目录，支持子文件夹批量处理
- `分析 Excel`
  - 导入现成 Excel 文件进行统计分析
- `一键保存`
  - 导出当前表格、图表和相关结果

### 2.2 视图区域

- `信息表格`
  - 查看明细记录和探头汇总
- `数据可视化`
  - 查看统计图表
- `表格 + 图表`
  - 同时查看表格和图表

### 2.3 历史功能

- 开启或关闭历史累计
- 切换查看当前批次
- 切换查看历史累计
- 查看历史详情

### 2.4 辅助功能

- 查看警告
- 查看去重信息
- 打开软件说明
- 打开数据安全提示

## 3. 常用操作流程

### 3.1 Excel 分析流程

1. 点击 `分析 Excel`
2. 选择 Excel 文件
3. 等待软件提取记录并生成统计
4. 在表格中查看明细和汇总
5. 在图表中查看可视化结果
6. 需要保存时使用 `导出` 或 `一键保存`

### 3.2 文件夹批量处理流程

1. 点击 `处理文件夹`
2. 选择本地或网络共享目录
3. 软件自动扫描满足条件的子目录
4. 按最小有效文件夹单位生成分析结果
5. 汇总显示到表格和图表中

### 3.3 历史累计流程

1. 保持历史记录功能开启
2. 每次分析完成后，结果可合并到本地历史累计
3. 切换到 `历史累计` 查看跨批次结果

## 4. 表格说明

### 4.1 探头使用信息表

主要字段包括：

- 大修
- 蒸汽发生器编号
- 数据组
- 操作员
- 探头类型
- 探头编号
- 探头型号
- 管道数量
- 累计管道数量
- 开始时间
- 结束时间

### 4.2 探头汇总统计表

按“探头编号 + 探头类型 + 探头型号”聚合，主要统计：

- 总使用次数
- 累计检测管道数量
- 最长连续使用
- 再次使用情况
- 检测速度

### 4.3 去重信息表

显示：

- 原始记录数
- 去重后记录数
- 移除重复数
- 被移除的重复记录详情

## 5. 图表说明

### 5.1 按探头编号统计

用于查看不同探头编号的使用情况，图表按探头类型、探头型号进一步区分。

常见图表包括：

- 使用次数图
- 累计检测管道数量图
- 检测速度图

### 5.2 按探头类型统计

用于查看探头类型整体分布和寿命差异。

常见图表包括：

- 探头类型分布
- 探头型号分布
- 不同批次平均寿命图

## 6. 关键统计规则

### 6.1 去重规则

只有当以下关键字段整体一致时，才会判定为重复记录：

- 大修
- SG 编号
- 数据组
- 操作员
- 探头类型
- 探头编号
- 探头型号
- 管道数量
- 开始时间
- 结束时间

### 6.2 寿命规则

探头寿命按“累计检测管道数量”统计，不以单纯使用时间作为寿命定义。

### 6.3 时间规则

时间主要用于：

- 判断记录是否有效
- 辅助识别连续使用
- 辅助识别再次使用
- 生成异常提示

### 6.4 连续使用判断

相邻记录是否属于同一连续使用段，会综合考虑：

- 时间间隔
- `SG_ID`
- 数据组编号连续性
- 中间是否出现其他探头占用

### 6.5 再次使用判断

如果同一探头被分成多个使用段，且中间存在其他探头插入使用，则判定为再次使用。

## 7. 文件夹解析规则

### 7.1 网络目录支持

支持读取网络共享目录或服务器映射盘中的数据，但按只读方式处理。

### 7.2 只读约束

程序不会对源目录做以下操作：

- 修改文件
- 删除文件
- 重命名文件
- 回写统计结果
- 回写缓存

### 7.3 特殊标记说明

`888`、`999` 等非真实检测标记，不计入真实管道数量统计。

## 8. 筛选与导出

### 8.1 筛选

支持：

- 表头下拉筛选
- 多选筛选
- 手动输入关键字筛选

切换筛选条件后，表格和图表会按当前条件联动刷新。

### 8.2 导出

支持导出：

- 当前表格
- 当前统计结果
- 单张图表
- 组合总图
- 一键保存全部结果

## 9. 异常与警告

软件会重点识别以下问题：

- 管道数量为 0
- 开始时间为空
- 结束时间为空
- 开始和结束时间都为空
- 结束时间早于开始时间
- 使用时长小于等于 0
- 数据组缺少关键标记

异常记录会在警告窗口中单独展示，便于复核。

## 10. 历史累计说明

历史累计数据保存在本地，用于跨批次分析，不影响原始数据源。

历史累计主要用于：

- 长周期使用趋势查看
- 跨批次寿命统计
- 重复导入后的去重合并

## 11. 推荐使用习惯

1. 优先分析本地副本或只读共享目录
2. 每次导入后先查看警告信息
3. 对关键结果及时导出归档
4. 发现异常统计时，先核对原始记录、去重信息和警告信息
5. 批量处理大目录时，耐心等待进度完成，不要连续重复触发同一操作

## 12. 上线前重点检查项

1. 启动后默认图表是否自动生成
2. 高频切换筛选是否还会出现空表
3. 一键保存和批量导出时是否有明确进度反馈
4. 弹窗是否支持重复打开、刷新内容和自由调整大小
5. 网络目录读取是否全程只读
