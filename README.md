# PIES 探头使用信息统计分析工具

PIES 是一个面向探头检测业务的数据整理与统计分析工具。当前仓库主要维护 PyQt5 桌面版实现，用于从 Excel 或业务数据文件中提取探头使用记录，并完成去重、统计、历史累计、图表展示、异常提示和结果导出。

## 项目用途

这个项目主要解决以下问题：

- 将分散的探头使用数据统一整理成结构化记录
- 统计单个探头的使用次数、累计使用时长和累计检测量
- 识别连续使用段与更换后再次使用情况
- 提供图表、表格和导出结果，方便复核和汇报

适合用于探头使用寿命、检测效率和历史记录的辅助分析。

## 主要功能

- 支持导入 Excel 数据进行统计分析
- 支持从业务数据文件中提取探头记录
- 按完整记录字段去重，避免重复统计
- 按“探头编号 + 探头类型 + 型号”聚合
- 统计使用次数、使用时长、最长连续使用时长、检测速度等指标
- 支持当前批次与历史累计数据切换查看
- 支持异常记录提示、日志记录和结果导出
- 支持图表展示与批量保存

## 运行环境

建议环境：

- Python 3.10 及以上
- Windows 桌面环境

主要依赖：

- PyQt5
- pandas
- openpyxl
- matplotlib
- numpy

安装依赖：

```bash
pip install -r requirements-pyqt.txt
```

## 启动方式

默认启动：

```bash
python run.py
```

等价入口：

```bash
python run_pyqt.py
```

程序启动后会自动创建 `logs/` 目录，并写入运行日志，方便排查异常。

## 统计口径说明

- 去重不是只按探头编号判断，而是按完整记录字段匹配
- 使用时长基于开始时间和结束时间计算
- 时间无效、时间异常、管道数量为 0 等记录会影响是否纳入有效统计
- 连续使用段识别会综合时间间隔、`SG_ID`、数据组编号等信息判断
