# PIES 探头使用信息统计分析工具

PIES 是一个用于整理和分析探头使用记录的桌面工具项目，当前仓库主要为 PyQt5 版本实现。项目面向探头检测业务场景，支持从 Excel 或相关数据文件中提取记录，完成去重、统计、历史累计、异常提示、图表展示和结果导出。

## 项目简介

这个项目的目标，是把分散的探头使用数据整理成可复核、可统计、可视化的分析结果，方便查看：

- 每个探头的使用次数
- 每个探头的累计使用时长
- 最长连续使用时长
- 更换后再次使用情况
- 累计检测管道数量
- 检测速度等统计指标

当前仓库中的主实现为 `pies_pyqt/`，提供完整的图形界面和核心分析逻辑。

## 主要功能

- 支持导入 Excel 数据并进行分析
- 支持解析业务数据文件并提取探头记录
- 按完整记录字段进行去重，避免重复统计
- 按“探头编号 + 探头类型 + 型号”聚合统计
- 识别连续使用段与更换后再次使用情况
- 展示图表和表格结果
- 支持历史数据累计分析
- 支持日志记录与异常提示
- 支持结果导出

## 项目结构

```text

  app.py                 PyQt 应用入口
```

## 运行环境

建议环境：

- Python 3.10 及以上
- Windows 桌面环境

主要依赖：

- PyQt5
- pandas
- openpyxl
- matplotlib
- numpy

安装依赖：

```bash
pip install -r requirements-pyqt.txt
```

## 启动方式

默认启动方式：

```bash
python run.py
```

也可以使用：

```bash
python run_pyqt.py
```

启动后会创建 `logs/` 目录，并将运行日志写入日志文件，便于排查问题。

## 统计口径说明

项目中的统计逻辑主要包括：

- 去重基于完整记录字段匹配，不是只按探头编号去重
- 使用时长基于开始时间和结束时间计算
- 无效时间、异常时间、管道数为 0 等记录会影响是否纳入有效统计
- 连续使用段识别不仅看时间间隔，也会结合 `SG_ID`、数据组编号等信息判断

如果你需要更细的业务规则说明，可以查看：

- `pies_pyqt/resources/software_manual.md`
- `pies_pyqt/core/`
