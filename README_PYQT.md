# PIES PyQt5 迁移版

## 说明
- 新代码位于 `pies_pyqt/`
- 默认启动入口为 `run.py`（等价于 `run_pyqt.py`）
- 依赖文件为 `requirements-pyqt.txt`
- Qt 业务核心位于 `pies_pyqt/core/`

## 目标
- 使用 PyQt5 提供主界面和业务工作台
- 使用 `QThread` 避免耗时任务阻塞界面
- 增加统一异常处理与可恢复提示

## 当前状态
这是一个可运行的 PyQt5 迁移版本，已包含独立核心逻辑、资源文件、历史数据兼容、快速筛选、表头筛选、图表、警告详情、历史累计、导出与一键保存等主要工作台能力。
